FRENCH / FRANCAIS :

Il s'agit ici d'un simple compteur de voyelles et de consonnes.

Vous indiquez au programme un mot et il vous dit le nombre de voyelles et de consonnes dans ce mot.

voici comment lancer le programme :

1) téléchargez le repos
2) rendez vous dans le dossier du compteur
3) lancer un cmd
4) tapez "python v_and_c_counter.py" ou "py v_and_c_counter.py" cela dépend de la version de python




ENGLISH / ANGLAIS :

This is a simple vowel and consonant counter.

You tell the program a word and it tells you the number of vowels and consonants in that word.

here is how to run the program:

1) download the rest
2) go to the counter folder
3) launch a cmd
4) type "python v_and_c_counter.py" or "py v_and_c_counter.py" it depends on python version
